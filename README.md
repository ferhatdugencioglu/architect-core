# 🏗️ Architect Core

> Architect Core is a tool I built to keep my technical notes organized. It is fully offline and can be kept in a single HTML file.

## What is it?

Architect Core is a personal tool I use to keep technical notes, commands, and scripts in one place. It also includes a Case Library for documenting support cases with a clear Problem → Root Cause → Solution flow.

There is no installation or backend involved. You just open the HTML file in your browser and start using it.

## Features

- 📝 **Tech Notes** — Keep commands, scripts, and notes organized by category
- 📚 **Case Library** — Document support cases with timeline and work log
- 🔐 **Local Encryption** — Data is encrypted locally in the browser
- 🌐 **Fully Offline** — Works without an internet connection
- 💾 **Backup & Restore** — Export and import data as JSON
- 🔍 **Global Search** — Search across notes quickly

## How to Use

1. Download `index.html`
2. Open it in Chrome, Edge, or Firefox
3. Create your account
4. Start adding your notes

## Tech Stack

- Vanilla JavaScript
- Web Crypto API
- IndexedDB + localStorage
- No external dependencies
